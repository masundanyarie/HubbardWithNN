BoseHubbard模型の基底状態と時間発展を機械学習によって求めるプログラム

bosehubbard1.cpp:エネルギーを最小化することによって基底状態を求める。
bosehubbard1.cpp:シュレディンガー方程式に基づいて基底状態や時間発展を求めるプログラム
overlap.cpp : 波動関数間のオーバーラップを最大化してネットワークを同期するプログラム。
　　　　　　　単純なコピーで無く、同じ出力をする別のネットワークを作る。

仕様：1次元・2次元のBoseHubard、FermiHubbardの基底状態、時間発展を求めるプログラム。
　　　FermiHubbardの基底状態はローカルミニマムに引っかかる可能性あり。
　　　BoseHubbardの時間発展は相互作用Uが大きいとうまくいかない。
